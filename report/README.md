# Power BI Report

After building the dashboard following `docs/POWERBI_BUILD_GUIDE.md`, save your file here as:

```
report/Zegna_Luxury_Dashboard.pbix
```

Commit the `.pbix` to git so anyone who forks the repo can open and adapt it in Power BI Desktop.

> 💡 `.pbix` files are binary but typically only a few MB — safe for git.
